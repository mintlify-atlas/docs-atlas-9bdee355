## What's included

The Wedding Website template is a single-page HTML/SCSS/JavaScript website with everything you need to invite, inform, and RSVP your guests.

<CardGroup cols={2}>
  <Card title="RSVP to Google Sheets" icon="table">
    Guests submit their name, email, and party size. Responses go directly into a Google Sheet — no server required.
  </Card>
  <Card title="Add to Calendar" icon="calendar">
    After RSVPing, guests can add the event to Google Calendar, Apple Calendar, Outlook, or Yahoo.
  </Card>
  <Card title="Google Maps venue" icon="map-pin">
    An embedded interactive Google Map shows guests exactly where the venue is located.
  </Card>
  <Card title="Book Uber" icon="car">
    A deep-link button lets guests book an Uber to the venue with a single tap on mobile.
  </Card>
  <Card title="YouTube video" icon="video">
    Embed a YouTube video that plays as a background showcase — great for a city or venue highlight.
  </Card>
  <Card title="Engagement photos" icon="image">
    A lightbox photo gallery with smooth hover overlays shows off your engagement pictures.
  </Card>
  <Card title="Invite code protection" icon="key">
    Protect your RSVP form with a numeric invite code — only guests with the code can submit.
  </Card>
  <Card title="Fully responsive" icon="mobile">
    Looks great on desktop, tablet, and mobile with animated scroll waypoints and a hamburger nav.
  </Card>
</CardGroup>

## Technology overview

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styles | SCSS (compiled via Gulp + gulp-sass) |
| Scripts | jQuery, Waypoints, FancyBox, FlexSlider |
| RSVP backend | Google Apps Script → Google Sheets |
| Maps | Google Maps JavaScript API |
| Calendar | ouical.js |
| Ride booking | Uber deep-link URL scheme |
| Build tool | Gulp 4 |
| Hosting | GitHub Pages (free) |
